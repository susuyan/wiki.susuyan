# 前端相关工具与组件库链接概要
## UIKit
- [shadcn/ui](https://ui.shadcn.com/) - 设计精美、可访问的组件及代码分发平台，兼容主流框架，开源开放。
- [Dice UI](https://www.diceui.com/) - 基于React、TypeScript和Tailwind CSS构建的shadcn/ui衍生组件库，支持自定义。
- [Radix UI](https://www.radix-ui.com/) - 优化开发效率与可访问性的开源组件库，无需配置直接导入使用。
- [Lucide](https://lucide.dev/) - 社区打造的轻量化、可扩展图标库，支持主流包管理器，自定义性强。
- [React Hook Form](https://react-hook-form.com/) - 高性能无依赖表单库，基于HTML标准，验证易用且挂载速度快。
- [Zod](https://zod.dev/) - TypeScript优先的模式验证库，支持静态类型推断，零依赖且API简洁。
- [Supabase UI](https://supabase.com/ui) - 基于shadcn/ui的React组件集合，可快速连接Supabase后端。
- [ved-ui](https://github.com/azacdev/ved-ui) - 基于Next.js的应用，整合radix、tailwind等技术，含完整开发目录结构。
- [Tailwind CSS](https://tailwindcss.com/) - 实用优先的CSS框架，支持响应式设计、暗模式，可直接通过预定义类构建网站。
- [Karrix](https://karrix.dev/) - 基于shadcn/ui和Motion的动画UI组件库，适用于项目快速集成。
- [Matsu-theme](https://matsu-theme.vercel.app/) - 受吉卜力工作室启发的shadcn/ui主题，含独特调色板与纹理设计。
- [NativewindUI](https://nativewindui.com/) - 基于React Native、Expo的移动组件库，提供30+原生质感组件与模板。
- [Fluid Design](https://fluid-design.io/) - 支持暗模式与可访问性的React UI组件库，具有流畅过渡动画。
- [Animate UI](https://animate-ui.com/) - 专注于动画效果的UI组件库，提供各类交互动画解决方案。
- [Nativewind](https://www.nativewind.dev/) - 允许在React Native中使用Tailwind CSS，支持跨平台一致样式设计。

## Template
- [Prism UI Template](https://templete-prismui.vercel.app/) - 基于Prism UI构建的前端模板，适用于快速搭建项目基础架构。
