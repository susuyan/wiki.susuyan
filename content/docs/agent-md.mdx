---
title: Agent 的一些 Markdown 文件
---

本页归档与 Agent 相关的 Markdown 文档，介绍各自的概念、用途及基本范式，便于快速查阅和复用。

---

## Claude.md

**概念**：记录 Claude 系列 Agent 的配置、用法和最佳实践。
**用途**：用于归档 Claude 智能体的 prompt、参数、API 使用方式等。
**基本范式**：

- 标题（如“Claude Prompt 模板”）
- 概念说明
- 用法示例
- 配置参数
- 注意事项

---

## Agent.md

**概念**：通用 Agent 的定义、核心能力和设计思路。
**用途**：总结 Agent 的基本架构、常用模式、交互流程等。
**基本范式**：

- Agent 概述
- 能力清单
- 工作流范例
- 交互流程
- 扩展方式

---

## SKILL.md

**概念**：Agent 可用技能模块的归档与说明。
**用途**：详细描述每个技能模块的功能、接口和用法。
**基本范式**：

- 技能名称
- 功能描述
- 接口定义
- 使用场景
- 示例代码
