# Inspiration

记录脑子里的胡思乱想灵感，目的清空大脑中的想法，找到落地可执行的有效正反馈循环。

基本灵感构建就是，以我想做个什么东西开头，然后根据这个东西，思考它的实现方式，找到落地可执行的有效正反馈循环。

## 💡 我的想法

我想做一套 iOS Native App（适用于 UIKit/SwiftUI） 基建平台，可以快速启动一个新的 iOS 项目。

- 最佳实践，阅读感友好的工程目录结构
- 标准ci/cd，自动化构建、测试、部署
- 网络层交互
- 多语言
- 使用 TCA 架构
- 多平台构建支持（iOS/MacOS/iPadOS）
- 使用 Swift Package，来进行组件化
- 好用的内购组件支持，订阅/买断, 内购状态管理, 使用revenuecat SDK
